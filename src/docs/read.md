## 项目简介

本项目用于学习three.js

## 技术栈

项目采用 vue3.x + vite2.x + element-plus1.x + typescript

## 目录说明

|          目录           | Description   
| ---------------------- | ------------------------------------
| `./src/docs`           | 文档目录，包含项目架构，以及threejs使用纪录
| `./src/views`          | 使用示例文档，按照day-num方式组织目录
